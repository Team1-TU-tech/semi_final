# MARIADB

## USAGE
```
$ git clone

# DOCKER RUN
$ docker compose up -d
$ docker ps

# CONTAINER
$ docker exec -it tut-mariadb bash
$ mariadb -u tut -p<password>
$ show databases;
$ use ticket;
```
- `.env` 파일에서 user id 및 password 확인 가능
